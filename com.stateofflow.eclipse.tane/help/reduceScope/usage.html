Reduce Scope - Usage

Given the code at the start of the above example, place the cursor within the local variable to be refactored (or select the variable name) in the Java editor:

double sum;
double[][] rows = getRows();
double[] means = new double[rows.length];

for (int row = 0 ; row < rows.length ; row++) {
    sum = 0;
    for (double col : rows[row])
        sum += col;
    means[row] = sum / rows[row].length;
}

Now invoke Quick Assist and select Reduce Scope. If the Reduce Scope Quick Assist is not available, it is because the preconditions for applying this refactoring have not been met.

The refactoring will work out what needs to be done and then show you a preview of the refactored code, as usual. If you are happy with the changes, click 'Finish' and they will be applied.
