Unify Declarations - Preconditions

  • The selected variable must be declared within a conditional statement (if or switch). The conditional does not have to be the immediate parent of the variable declaration.
  • A variable with the same name as the selected one must be declared in at least one other branch of the conditional, and all such variables must have the same type as the selected variable.
  • No variable with the same name may be declared after the conditional statement within the parent scope of the conditional.
  • No references to any of the variables (whose declarations are to be unified) must require the final modifier in the declaration, i.e. no references must be made from within a local class.

If you find that the refactoring fails under conditions that should be checked as a precondition, please get in touch.
