Hide Delegate - Example

Suppose you have code that looks like this:

Foo foo = new Foo(); … foo.getBar().getBaz().doSomething();

In order to move to the OO ideal of eliminating state based metaphors and focusing on message passing, we need to turn this code into something more like this:

Foo foo = new Foo(); … foo.doSomething();

One way of achieving this is to implement Foo.doSomething() like this:

public class Foo { … public void doSomething() { getBar().getBaz().doSomething(); } … }

Clearly the same process can be followed to push the behaviour further into the model so that Foo.doSomething() is implemented as getBar().doSomething().
