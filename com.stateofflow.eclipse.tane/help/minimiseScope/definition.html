Minimise Scope - Definition

In Refactoring: Improving the Design of Existing Code ( ), Martin Fowler describes the problem and solution for the Hide Delegate refactoring thus (pp. 157):

A client is calling a delegate class of an object.
 
Create methods on the server to hide the delegate.

Here are some class diagrams showing the situation before and after the refactoring is applied.

The effect of this is to reduce coupling, abstract behaviour, reduce the use of state based metaphors and focus on message passing by pushing a Demeter violating chain of method calls into a more appropriate class.
