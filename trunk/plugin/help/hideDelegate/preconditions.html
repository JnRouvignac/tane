Hide Delegate - Preconditions

  • All elements in the chain must be method invocations, except for the last which may be a field access.
  • The source must be available for the receiver of the first method invocation.
  • The receiver of the first invocation must be a class, not an interface.

If you find that the refactoring fails under conditions that should be checked as a precondition, please let me know via the website.
